# LifeCadenceLedger

[![Status](https://img.shields.io/badge/status-early_development-orange?style=flat-square)](#)

> A personal ledger for the rhythms, habits, and obligations that structure daily life.

LifeCadenceLedger tracks recurring commitments — the cadence layer that sits between a calendar and a habit tracker. Project structure is being established.

## Planned Features

- Recurring commitment tracking with configurable cadences
- Habit and obligation ledger with historical view
- Daily rhythm visualization
- Local-first — no accounts, no cloud required
