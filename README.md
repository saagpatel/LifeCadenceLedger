## Getting Started

```bash
# Clone the repository
git clone https://github.com/saagpatel/LifeCadenceLedger.git
cd LifeCadenceLedger

# Install dependencies (once package.json is present)
npm install

# Run in development mode
npm run dev

# Build for production
npm run build
```
